
# RANAssist AI

AI Powered Telecom RAN Assistant using:
- React
- Flask
- Groq LLM
- RAG
- FAISS
- Sentence Transformers

================================================

## FRONTEND SETUP

cd frontend

npm install

npm start

================================================

## BACKEND SETUP

cd backend

pip install -r requirements.txt

python app.py

================================================

## GROQ API KEY

Open:

backend/app.py

Replace:

YOUR_GROQ_API_KEY

with your actual Groq API key.

================================================

## FEATURES

- RAG Based Retrieval
- Telecom Knowledge Base
- AI Answer Generation
- React Frontend
- Flask Backend
- Groq LLM Integration
- FAISS Vector Search
